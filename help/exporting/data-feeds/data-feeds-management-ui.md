# Create and manage [!UICONTROL data feeds] {#data-feeds-management-ui}

The Adobe Analytics [!UICONTROL Data feed] allows you to export raw clickstream data to import and merge with other data sources outside of Adobe Analytics. You can create and manage your own [!UICONTROL data feeds] within Adobe Analytics. This video will show you how.

For more information on this feature, please see the [documentation](https://experienceleague.adobe.com/docs/analytics/export/analytics-data-feed/df-manage-feeds.html?lang=en#).
